> 开始使用 Quicker 剪贴板前，你只需要完成安装与基本设置。下面带你在数分钟内上手。
> 目前仅支持 Quicker 中使用，暂时没有独立软件版本。

## 安装

1. 安装 [Quicker](https://getquicker.net/Download)，按照安装向导完成安装。
2. 安装 Quicker 剪贴板插件，复制链接然后在quicker面板空白处点击安装

## 运行

1. 点击面板上的动作，（首次启动加载各种配置服务，需要等待3~5秒，具体时间和电脑配置有关）

2. 如果你使用的是quicker专业版，可以将动作添加到[自动运行](quicker:settings:AutoRunActions)。

## 使用

1. 复制文本、图片、文件路径等。

2. 点击面板上的动作，在历史中找到刚刚的内容，试试搜索、收藏与粘贴。

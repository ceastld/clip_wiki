# 捷径面板3.0

捷径面板3.0是一个强大的剪贴板动作管理工具，提供直观的面板界面来快速访问和管理你的收藏动作。

## 功能特性

- 📋 **数据来源**：面板中的数据来源于剪贴板动作的收藏夹
- 🖱️ **双击返回**：进入收藏夹后，双击空白位置可以返回上级
- 🎯 **右键菜单**：更多功能可通过右键菜单访问
- 🎨 **自定义背景**：支持暗黑透明背景设置
- ⚡ **快速访问**：提供便捷的动作调用方式

## 使用方法

### 基本操作

1. **打开面板**：点击捷径面板按钮或使用快捷键
2. **浏览动作**：面板显示收藏夹中的所有动作
3. **执行动作**：点击动作项即可执行对应功能
4. **返回上级**：双击空白位置返回上一级目录

### 右键菜单功能

右键点击面板空白处或动作项，可以访问更多功能：

- 添加新动作到收藏夹
- 编辑现有动作
- 显示动作的完整右键菜单
    * 包括quicker动作的默认右键菜单和自定义的右键菜单
- 调整面板设置

## 自定义设置

### 背景色设置

在设置中，你可以调整面板的背景色：

- **暗黑透明背景**：设置为 `#53000000`
- **透明度控制**：调色框最下面的 A 值表示透明度
- **实时预览**：调整后立即生效

### 设置步骤

1. 右键点击面板空白处
2. 选择"设置"选项
3. 找到"背景色"设置项
4. 调整颜色值，注意 A 值控制透明度
5. 点击确认保存设置

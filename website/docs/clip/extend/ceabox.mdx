# CeaBox

CeaBox 是一个强大的剪贴板管理工具，提供搜索框、收藏夹、剪贴板聚合等功能。

## 功能特点

- **搜索框**: 快速搜索剪贴板内容
- **收藏夹**: 管理常用文本和文件
- **剪贴板聚合**: 统一管理多个剪贴板
- **捷径面板**: 快速访问常用功能
- **键盘操作**: 支持快捷键操作

## 使用说明

### 准备工作

1. 安装剪贴板动作
2. 更新quicker到1.31以上版本

### 主要功能

1. **搜索框**: 输入关键词快速搜索剪贴板内容
2. **收藏夹**: 右键收藏常用内容到指定文件夹
3. **捷径面板**: 快速访问常用功能和文件
4. **键盘操作**: 支持ESC隐藏、方向键选择等

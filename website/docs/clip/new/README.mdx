# 最新动态


## 最新版更新后可能遇到的问题

### 收藏夹数据好像没了？

更新动作后，收藏夹数据如果遇到没有成功导入的情况，则需要手动进行导入

import test from './image/test.png';
import { ClickableImage } from '@site/src/utils/mdx-imports';

<ClickableImage src={test} alt="测试" />

也可能是创建了空白收藏夹，需要点击左上角的返回按钮，放回上级进入正确的收藏夹。

### 置顶内容找不到了？

更新后，置顶内容改为了分组显示，可以点击分组名称进行切换。

如果想要改回之前的置顶效果，需要打开设置界面修改对应的筛选器配置

import top from './image/top.png';

<ClickableImage src={top} alt="置顶内容分组" />

> 开始使用 Quicker 剪贴板前，你只需要完成安装与基本设置。下面带你在数分钟内上手。

## 安装

1. 下载并解压安装包。
2. 按照安装向导完成安装。
3. 首次启动后，会在系统托盘常驻。

## 启动与唤出

- 默认支持通过热键唤出主界面（你可以在偏好设置中修改）。
- 首次唤出会创建必要的本地数据目录。

## 快速上手

1. 复制任意文本/图片/文件路径。
2. 打开 Quicker 剪贴板界面。
3. 在“历史”中找到刚刚的内容，试试搜索、收藏与粘贴。

## 常见问题

- 看不到界面？检查是否被最小化到托盘，或热键冲突。
- 没有记录？确保复制操作来自系统剪贴板，且应用有权限访问。

如需更完整的引导，可前往 `clip/快速开始` 与 `使用指南` 部分继续阅读。
